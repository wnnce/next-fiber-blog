# next-fiber-blog
一个博客系统，基于多种技术栈实现

- 后端：`go-fiber` + `sonic` + `pgx` + `redis` + `elasticsearch` + `jwt`
- 管理后台：`Vue3` + `Pinia` + `Typescript` + `ArcoDesign`
- 前端：`Next.js` + `Typescript` + `UnoCSS`
